# 요리 레시피 검색 앱

## 1. 어플 기능

**검색**
- 검색을 통한 메뉴 레시피 검색
- 검색 결과를 리스트로 표시

**레시피 상세 보기 화면**
- 메뉴 이미지, 필요한 재료, 조리법 등 상세 정보 제공
- 스크롤 형식으로 상세 화면 구성

**즐겨찾기 기능**
- 마음에 드는 레시피 즐겨찾기로 저장
- 즐겨찾기 탭에서 저장한 레시피 열람 가능
- 즐겨찾기 추가 및 삭제 기능

**API연동**
- 레시피 API를 통한 메뉴 검색
- 사용 API : https://www.foodsafetykorea.go.kr/api/openApiInfo.do?menu_grp=MENU_GRP31&menu_no=661&show_cnt=10&start_idx=1&svc_no=COOKRCP01
