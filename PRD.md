# 1. 프로젝트 개요 
- 재료 / 키워드 기반 요리 레시피 검색 iOS 앱
- 사용 언어 : SwiftUI
- 구조 : MVVM 구조 
- API 사용
- 로컬 저장

## 2. 타겟 사용자 
- 20~30대 자취생
- 30~50대 주부
- 요리 관련 취미 / 직업 보유자
- 매일 먹을 요리가 고민되는 사람

## 공통 니즈 
- 가진 재료를 활용하는 메뉴를 빠르게 찾고 싶음
- 조리 시간, 난이도에 맞는 레시피를 쉽게 고르고 싶음 
- 마음에 드는 메뉴를 즐겨찾기해서 반복해서 참고하고 싶음

### 3. 핵심가치 제안

# 재료기반 레시피 검색
- 냉장고에 있는 재료를 입력하면 그 재료들로 만들 수 있는 레시피 추천

# 간단한 필터 (시간 / 난이도 등)
- 바쁜 일상에 맞춰 조리시간, 요리 실력에 맞춰 난이도로 결과를 좁혀서 보여줌

# 즐겨찾기 기능
- 자주 해먹는 메뉴를 즐겨찾기에 저장
- 다음에 다시 찾을 때 검색 과정 없이 바로 접근 가능

### 4. 사용 시나리오 
# 가지고 있는 재료를 통한 레시피 검색 
- 사용자가 가진 재료 (계란, 양파, 햄 등) 재료를 검색창에 입력
- 검색 버튼 또는 자동 검색을 통한 API 호출
- 조건에 맞는 레시피 목록을 리스트 / 격자로 표시

# 레시피 상세 보기 
- 검색 결과 중 하나를 탭
- 해당 레시피의 이미지, 요리 이름, 재료 목록, 조리 순서를 확인

# 레시피 즐겨찾기 등록
- 상세화면 또는 리스트 아이템에서 즐겨찾기 아이콘을 탭하여 추가/삭제

# 조리시간/난이도 필터링

# 기능 요구사항
### 레시피 검색 기능
- 사용자에게서 레시피 검색 입력을 받고 검색 결과를 출력함
- 앱은 외부레시피 (API)를 호출해 검색 결과 가져옴
- 검색 결과는 리스트로 표시 
 
### 레시피 상세 화면
- 리스트 아이템을 탭하면 상세 화면으로 이동 
- 상세 화면에는 다음과 같은 정보 포함
1) 레시피 이름
2) 대표 이미지 
3) 재료 목록
4) 조리 순서

### 즐겨찾기
- 상세화면, 리스트 화면에서 즐겨찾기 추가/삭제 버튼 제공
- 즐겨찾기 레시피는 로컬 저장소에 저장
- 즐겨찾기 탭에서 저장된 레시피 목록 열람 가능 

# 개발 관련사항

## 개발환경
- Xcode 16.x
- iOS 16 이상

## 코드 스타일 및 구조 
- Model / View / View Model 구조로 디렉터리 분리 


# UI / UX
색상 : 흰색 계열 바탕
타이포 : 가시성 우선
인터랙션 : 스와이프 제스쳐

## 핵심화면 
홈 : 최상단 어플 이름 + 상단 검색바 + 중단 오늘의 추천 메뉴(레시피 포함) + 하단 탭 바 
검색화면 : 키워드 / 재료명을 통한 메뉴 자동 서칭 + 최근 검색어 노출 
검색 결과화면 : 결과 레시피 이미지 + 메뉴 이름 출력. 격자형식
레시피 화면 : 메뉴 이미지, 이미지 아래 메뉴 이름 + 레시피 재료 텍스트 + 레시피 조리법 순서 VStack 

### AI 담당 업무
1) API 적용
- 레시피 API를 적용하고 검색 리스트, 상세 페이지에 출력
2) 오류 분석 및 리팩토링 아이디어
- 컴파일 에러, 런타임 에러 메세지를 통한 문제 유추 및 해결방안 제안
- 작성한 코드를 통해 네이밍/구조/중복 코드에 대한 리팩토링 아이디어 제안
